Dark Star: APN Setup

Follow these steps to configure the APN settings on your Android device:

  1. Go to Settings > Connections / Network & Internet > Mobile Networks / Cellular Networks > Access Point Names. Some phones may have this option under:

    • Settings > Network & Internet > Mobile Network > Advanced > Access Point Names
    • Settings > Mobile Network > Access Point Names

  2. Tap on + Add or the Menu icon (three dots) and select New APN.

  3. Enter the following details:

    • Name: Dark Star
    • APN: ERESELLER
    • MMSC: http://mmsc.mobile.att.net
    • MMS Proxy: proxy.mobile.att.net
    • MMS Port: 80
    • MCC: 310
    • MNC: 410 (280 if the APN does not save).
    • APN Type: default,supl,mms,fota
    • APN Protocol: IPv4/IPv6

    4. ‎ -‎ ‎ ‎ ‎ Leave any unlisted fields blank.

  4. Tap Save or ✓.

  5. Ensure that the new APN is selected and confirm if your APN is saved.

Hotspot

If you are receiving an error when enalbing hotspot, you will need to disable "Private DNS".

Go to Settings > Network & Internet > Private DNS under security settings.

Turn this OFF and try to enable Hotspot.

w/ Proxy: Need Help?

What is W/ Proxy?

"W/ Proxy" stands for "With Proxy." This option allows you to connect to the internet through a proxy server.

When Should You Use This Option?

Use this option if you need to route your internet connection through a proxy server, either for privacy or network configuration purposes.

How to Use It:

1. Click the "W/ Proxy" button.
2. Ensure that your proxy settings are configured properly in your network settings.

Troubleshooting:

If you encounter issues while using "W/ Proxy," check your proxy settings and ensure that the proxy server is functioning correctly.

w/o Proxy: Need Help?

What is W/O Proxy?

"W/O Proxy" stands for "Without Proxy." This option allows you to connect to the internet directly without using a proxy server.

When Should You Use This Option?

Use this option if you do not require a proxy server for your internet connection or if you are experiencing issues with proxy configurations.

How to Use It:

1. Click the "W/O Proxy" button.
2. Ensure that no proxy settings are configured in your network settings.

Troubleshooting:

If you encounter issues while using "W/O Proxy," ensure that no proxy settings are enabled in your network configuration.
